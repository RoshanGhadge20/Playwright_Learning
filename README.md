# 🎭 Playwright Learning – JavaScript Automation Framework

This repository contains all learning materials, practice scripts, and automation examples related to **Playwright** using **JavaScript**.  
It covers UI automation, API testing, framework setup, and advanced Playwright concepts.

---

## 📌 About Playwright

Playwright is an open-source automation framework developed by Microsoft for end-to-end testing of web applications.

It supports:

- ✅ Chromium
- ✅ Firefox
- ✅ WebKit  
- ✅ Cross-browser & cross-platform testing
- ✅ API Testing
- ✅ Network Interception
- ✅ Parallel Execution

---

## 🛠 Tech Stack

- **Language:** JavaScript (Node.js)
- **Framework:** Playwright Test
- **Assertion Library:** Built-in `expect`
- **Package Manager:** npm
- **Reporting:** HTML Report
- **Version Control:** Git & GitHub
